Flexbox Grid

  • EncoreUI includes a grid system forked from Angular Material's layout module . With minor usability enhancements, EncoreUI provides an assortment of attribute-based layout options, based on the flexbox layout model. Included are intuitive attribute based styles for responsive row and/or column based page layouts. Visit the layout section for info on how to use these attributes.

Date formatting

  • Dates in any context should be formatted this way: Mar 2, 2015 @ 15:40 (UTC-0600) — a three-letter capitalized month abbreviation (no period), followed by the day of the month (no leading zeroes), a comma, the four-digit year, an @ symbol, military time, and UTC in parentheses.
  • The only time this formatting standard would not be used is when relative dates would be helpful to the user, such as "4 hours ago" or "1 week ago".
  • It is recommended that relative dates be reserved for more informal settings like user notes or comments. Relative dates should not extend too far into the past. For instance, "1 week ago" may be helpful to the user, but "8 months ago" or "3 years ago" is better represented by the non-relative format detailed above, as it reveals more specific information about dates that provide better context and information to the user when comparing dates and times.

Helper Classes

  • To make text less prominent, change its color using the .subdued class. This change will still persist if the class is within a link.
  • You can mark content as full width. For example, styling a full width button by using the .full-width class. This class requires 'display' to be styled as inline-block or block.
  • Hidden content
    • To hide an element but not remove it from the page spacing, use the .hidden class.
    • You can hide content visually and from the page spacing (but can still be read by screen readers) using the .visually-hidden class.
  • Clearing floats
    • To clear child elements using clearfix, you can use the .clear class. Learn more about the hack .
    • To clear previous sibling floats, you can use the .clear-left and .clear-right classes.

Progress Bars

  • There are two states for progress bars: active and complete. Active progress bars have animated stripes, which are produced by the class .active. Full progress bars do not have this class, and are solid and static in appearance.
  • Several options for displaying text with a progress bar are available: no text, a percentage (not italicized), and detailed text (italicized). Always try to show a percentage.
