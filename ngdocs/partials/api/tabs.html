tabs Component

This component provides styles and a demo for the the Angular-UI Bootstrap Tabs plugin, which is included as a dependency for EncoreUI.

Usage

Usage is the exact same as demoed on the Angular-UI Bootstrap site. See the Angular-UI Bootstrap Docs for further guidance on usage and configuration of this component.

Disclaimer

Only the default horizontal tabs are supported by these styles. Vertical, Pills and Justified tabs are currently unsupported.
