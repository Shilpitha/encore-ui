rxStatusColumn
directive in module rxStatusColumn

Description

A directive for drawing colored status columns in a table. This takes the place of the for the column it's in.

Usage

as attribute
<ANY rx-status-column
     status="{String}"
     [api]="{String}"
     [tooltip]="{String}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
statusString

The status to draw

[api]String

Optionally specify which API mapping to use for the status

[tooltip]String

The string to use for the tooltip. If omitted, it will default to using the passed in status
