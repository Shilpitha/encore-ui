rxModalFooter
directive in module rxModalAction

Description

Define a footer for the next modal.

Usage

as element:
<rx-modal-footer
       state="{String}"
       [[global]="{String}"]>
</rx-modal-footer>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
stateString

The content will be shown in the footer when this state is activated.

[global]
(optional)
String

If the global attribute is present, then this footer can be used in other modals. This attribute takes no values.

Example

<rx-modal-footer state="confirm">
    <button class="button" ng-click="setState('pending')">I understand the risks.</button>
</rx-modal-footer>
