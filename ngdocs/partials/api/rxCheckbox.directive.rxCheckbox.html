rxCheckbox
directive in module rxCheckbox

Description

Attribute directive that wraps a native checkbox element in markup required for styling purposes.

Usage

as attribute
<ANY rx-checkbox
     [ng-disabled="{Boolean}"]>
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
ng-disabled
(optional)
Boolean

Determines if the control is disabled.

(default: false)

Example

<input rx-checkbox ng-model="demoValue" />
