UnauthorizedInterceptor
service in module rxUnauthorizedInterceptor

Description

Simple injector which will intercept http responses. If a 401 is returned, the ui redirects to /login.

Dependencies

  • $q
  • @window
  • Session

Example

angular.module('encoreApp', ['encore.ui'])
    .config(function ($httpProvider) {
        $httpProvider.interceptors.push('UnauthorizedInterceptor');
    });
