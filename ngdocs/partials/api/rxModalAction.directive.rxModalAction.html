rxModalAction
directive in module rxModalAction

Description

Link which will show a modal window on click, and handle callbacks for pre/post modal actions

Usage

as element:
<rx-modal-action
       [[pre-hook]="{Function}"]
       [[post-hook]="{Function}"]
       [[dismiss-hook]="{Function}"]
       [[template-url]="{String}"]
       [[disable-esc]="{*}"]>
</rx-modal-action>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
[preHook]
(optional)
Function

Function to call when a modal is opened

[postHook]
(optional)
Function

Function to call when a modal is submitted (not called when cancelled out of)

[dismissHook]
(optional)
Function

Function to call when a modal is dismissed (not called when submitted)

[templateUrl]
(optional)
String

URL of template to use for modal content

[disable-esc]
(optional)
*

If the disable-esc attribute is present, then "Press Esc to close" will be disabled for the modal. This attribute takes no values.

Example

<rx-modal-action
    pre-hook="myPreHook(this)"
    post-hook="myPostHook(fields)"
    template-url="modalContent.html"
    disable-esc>
        My Link Text
 </rx-modal-action>
