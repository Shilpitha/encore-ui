rxBulkSelect
directive in module encore.ui

Description

A directive you place in your element to enable bulk select. This directive will automatically add into your , which will appear whenever items are selected, and disappear when none are selected. The main responsibility of this directive is to provide a controller for other bulk-select-related directives to interact with.

Usage

as attribute
<ANY rx-bulk-select
     bulk-source="{Object}"
     selected-key="{string}"
     resource-name="{string}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters
ParamTypeDetails
bulkSourceObject

The source list that the table ng-repeats over.

selectedKeystring

The attribute on items in bulkSource that will be used to track if the user has clicked the checkbox for that item.

resourceNamestring

The name of the resource being iterated over.

(default: bulkSource)
