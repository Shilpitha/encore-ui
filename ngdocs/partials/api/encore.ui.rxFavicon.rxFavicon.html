rxFavicon
directive in module encore.ui

Description

Updates href of element and replaces it with path to a different image based on environment

Usage

as attribute
<ANY rx-favicon="{Object}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
rxFaviconObject
  • Takes two optional properties (staging and local), with the value of each being the path to the favicon.
