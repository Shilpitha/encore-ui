rxSelectFilter
directive in module rxSelectFilter

Description

Automatically creates the appropriate dropdowns to manage a filter object.

rxSelectFilter

Uses an instance of SelectFilter to create a set of <rx-multi-select>'s that modify the instance object. 

// In the controller
$scope.filter = SelectFilter.create({
  // options...
});

// In the template
<rx-select-filter filter="filter"></rx-select-filter>

Usage

as element:
<rx-select-filter
       filter="{Object}">
</rx-select-filter>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
filterObject
  • An instance of SelectFilter
