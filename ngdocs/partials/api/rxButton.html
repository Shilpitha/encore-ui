rxButton Component

rxButton is used to create buttons with a dynamically-displayed loading indicator. This is used as a replacement for <button> elements in scenarios where the button has multiple states.

Styling

There are several styles of buttons available, and they are documented in the Encore Framework Styleguide. Any classes that need to be added to the button should be passed to the classes attribute.
