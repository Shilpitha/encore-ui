rxMultiSelect
directive in module rxSelectFilter

Description

A multi-select dropdown with checkboxes for each option

Usage

as element:
<rx-multi-select
       ng-model="{String}"
       [options]="{Array}">
</rx-multi-select>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
ng-modelString

The scope property that stores the value of the input

[options]Array

A list of the options for the dropdown
