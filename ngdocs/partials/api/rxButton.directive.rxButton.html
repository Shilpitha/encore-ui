rxButton
directive in module rxButton

Description

Renders a button which will disable when clicked and show a loading message and renable when operation is complete. If you set classes attributes <rx-button>, those will get passed to the <button> instance as class

Usage

as element:
<rx-button
       loading-msg="{String}"
       default-msg="{String}"
       [toggle="{Boolean}"]
       [disable="{Boolean}"]
       [classes="{String}"]>
</rx-button>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
loadingMsgString

Text to be displayed when an operation is in progress.

defaultMsgString

Text to be displayed by default an no operation is in progress.

toggle
(optional)
Boolean

When true, the button will display the loading text.

(default: false)

disable
(optional)
Boolean

When true, the button will be disabled.

(default: false)

classes
(optional)
String

The class names to be applied to the button.

(default: "")
